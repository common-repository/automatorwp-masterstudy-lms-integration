Connect AutomatorWP with MasterStudy LMS

== Description ==

> **Important:** Now all free integrations are included in [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") so you no longer need to install them one by one!
>
> We will continue working to improve AutomatorWP and make it easier, faster and more accessible to everyone.

[MasterStudy LMS](https://wordpress.org/plugins/masterstudy-lms-learning-management-system/ "MasterStudy LMS") is a full-featured WordPress LMS plugin that allows you to easily create and sell courses online.

This plugin automatically connects [AutomatorWP](https://wordpress.org/plugins/automatorwp/ "AutomatorWP") with MasterStudy LMS adding new triggers and actions that you can use to connect with other plugins and automate workflows letting you save time and get focused on your most important work.

= Triggers =

* User attempts any/specific quiz.
* User completes any/specific lesson.
* User completes any/specific course.

= Actions =

* Enroll user to all/specific course.

= Pro version =

[Pro version](https://automatorwp.com/add-ons/masterstudy-lms/ "AutomatorWP - MasterStudy LMS") available that let's you expand the integration between AutomatorWP and LearnPress bringing you more ways to automatize your elearning website.

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
